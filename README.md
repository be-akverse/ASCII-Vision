## ASCII Vision

A web based tool that coverts your live camera feed into ASCII Art which is downloadable in ".txt" form!

## AI Usage
AI was used in this project to fix and rewrite parts of javascript code.

## Live Demo
[https://be-akverse.github.io/ASCII-Vision/](https://be-akverse.github.io/ASCII-Vision/)

## Features
- Turn Live camera feed into ASCII Art
- ASCII Art is downloadable
- 4 different ASCII Art themes to convert into

  ## Tech Stack
  - HTML
  - CSS
  - JS
